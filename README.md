# openpdc-demo

## Requirements:
- docker
- docker-compose
- ddev

## install
clone the project
```
git clone git@github.com:Richard-NL/openpdc-demo.git
```

Start ddev and install dependencies
```
ddev start
ddev composer install
```

## Configure site:
https://openpdc.ddev.site

## Activate the plugins
Go to https://openpdc.ddev.site/wordpress/wp-admin/plugins.php
Active Meta Box Lite first
Now activate Yard|PDC Base
